# CGB++
CGB++ is a Gameboy original(the one that use DMG) Emulator

# Features
- ✅ Shows graphics
- ✅ Input implemented
- ✅ All CPU instructions implemented
- ✅ Debugger Built-In
- ✅ SDL2 and OpenGL
