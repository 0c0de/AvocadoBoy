# AvocadoBoy
AvocadoBoy is an emulator of the original Gameboy(DMG)

# Features
- ✅ Implemented graphics
- ✅ Implemented Input
- ✅ All CPU instructions implemented (CB Opcodes included)
- ✅ Implemented timers
- ✅ Implemented Interrupts
- ✅ Small Debugger Built-In
- ✅ SDL2 and OpenGL
